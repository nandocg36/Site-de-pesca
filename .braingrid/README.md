# BrainGrid neste repositório

Este projeto está preparado para integração com [BrainGrid](https://docs.braingrid.ai).

1. Instalar a CLI: `npm install -g @braingrid/cli`
2. Autenticar: `braingrid login`
3. Associar o repositório a um projeto: `braingrid init`

Após `braingrid init`, será criado `.braingrid/project.json` com a ligação ao projeto na plataforma.

Regras do Cursor: `.cursor/rules/braingrid.mdc` (sempre ativas).
